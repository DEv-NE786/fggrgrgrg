General Commands #1
  • -bot ↠ To show Bot statues
  • -user ↠ To show user info
  • -avatar ↠ To show member Avatar
  • -color ↠ To Pick a color in the server (If turned on by server admin)
  • -credits ↠ To show the amount of credits you have
  • -daily ↠ To take your daily gift
  • -inf ↠ To show invites infos
Admin Commands #2
  • -clear ↠ To clear the chat
  • -ban ↠ To ban a member
  • -kick ↠ To kick a member
  • -close ↠ To close the chat
  • -open ↠ To open the chat
  • -move ↠ To to move a member
  • -reply ↠ To create a reply command
  • -say ↠ To let the bot say what you are saying
  • -setLog ↠ To set a log channel (for safety reasons)
  • -setby ↠ To set a channel to goodby members who quit the server
  • -setwelcomer ↠ To set a channel to welcome new members
  • -ls ↠ To show all bots in the server
  • -role ↠ To give a member a certain role
  • -roleall ↠ To give a;; the members a certain role
